Orquestraio v0.4

Uma integração.
Múltiplos gateways.

O Orquestraio é uma plataforma de orquestração de pagamentos multi-tenant. Integre Stripe, MercadoPago e PayPal de uma vez — com roteamento inteligente, idempotência e resiliência embutidos.

Primeiros Passos Ver API
API

Endpoints & Recursos

Processe pagamentos, configure roteamento inteligente e receba notificações em tempo real.

Processar Pagamento

Cria uma transação e roteia automaticamente para o gateway ideal com base na moeda e método de pagamento.

Métodos de Pagamento

PIX, Boleto e Cartão via paymentMethodRequest.type. Suporte nativo a múltiplos gateways.

SmartRouter

Roteamento automático por moeda e valor. Sobrescreva por tenant via metadata.gateway.

Webhooks

Notificações assíncronas normalizadas cross-gateway. Implemente uma vez para todos os provedores.

Erros & Status

Códigos HTTP padronizados com timestamp, status, error, message e path.

Rate Limiting

Limites por tenant com Bucket4j + Redis. Proteção contra abuso e DDoS em nível de aplicação.
Funcionalidades

Resiliência e consistência por padrão

Idempotência, circuit breaker e criptografia fazem parte do core do Orquestraio — não são opcionais.

Idempotência

Proteção contra cobranças duplicadas via Redis com TTL de 24h. Mesma chave, mesmo resultado.

Circuit Breaker

Cada gateway tem circuit breaker independente via Resilience4j. Falhas são isoladas com fallback.

Segurança

Credenciais de gateway criptografadas com AES-256. Dados de cartão nunca passam pelo servidor.
Guias

Comece agora

Do zero ao primeiro pagamento aprovado em 3 minutos.

Primeiros Passos

Setup completo em 3 minutos usando Docker e gateway MOCK. Sem credenciais necessárias.

Autenticação

API Key via header HTTP x-api-key. Chaves de produção com prefixo Oz_.

Sandbox & MOCK

Simule cenários reais sem usar dinheiro real. Gateway MOCK disponível para qualquer tenant.
