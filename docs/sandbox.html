Sandbox & Gateway MOCK

2 min de leitura

Simule cenários reais sem usar dinheiro real. O gateway MOCK está disponível para qualquer tenant em desenvolvimento.

Chave de dev inclusa: O data.sql já cria um tenant com API key orquestraio-livre-123. Use direto após docker-compose up. Nenhuma credencial externa necessária.

Comportamento do Gateway MOCK

Valor (amount)ResultadoUso
Qualquer (exceto 99.99)APPROVEDFluxo normal de aprovação
99.99ERRORSimula falha no gateway

O gateway MOCK simula 300ms de latência para aproximar do comportamento real de gateways externos.
